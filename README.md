**# 🚀 ShelbyHub

🌐 Live Demo: https://shelbyhub-roan.vercel.app/

ShelbyHub is a modern web-based platform designed as an all-in-one hub for personal branding, file sharing, and digital tools exploration in one place.

Built with the concept of link-in-bio + Web3 storage + showcase platform, ShelbyHub makes it easy for users to manage, store, and share content efficiently.

---

## ✨ Features

- 🔗 Link-in-Bio Hub  
  Collect all your important links in a single page.

- 📦 File Sharing  
  Upload files and share them easily via link.

- 🌐 Web3 Storage (Planned/Integrated)  
  Secure and scalable decentralized storage.

- 🧩 Modular System  
  Easily expandable into a larger ecosystem.

- ⚡ Fast & Responsive UI  
  Built with modern technologies for optimal performance.

---

## 🛠️ Tech Stack

- Frontend: Next.js / React  
- Styling: Tailwind CSS  
- Deployment: Vercel  
- Future Integration: Web3 (Shelby Protocol / Aptos)

## 🚀 Getting Started

Clone the repository:

git clone https://github.com/Tezqee/shelbyhub.git  
cd shelbyhub  

Install dependencies:

npm install  

Run the development server:

npm run dev  

Open in your browser:

http://localhost:3000
