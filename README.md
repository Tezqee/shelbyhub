# 🚀 ShelbyHub

> **Link-in-bio + File Sharing + Web3 Storage Platform** powered by Shelby Protocol & Aptos

ShelbyHub is a modern platform that combines:
- 🔗 Link-in-bio (like Linktree)
- 📂 File sharing
- ☁️ Web3 decentralized storage

Built for creators, developers, and Web3 users to **store, share, and manage content in a decentralized way**.

---

## ✨ Features

- 🔗 Custom Link-in-Bio Page  
- 📁 Upload & Share Files via Link  
- 🌐 Web3 Storage Integration (Shelby Protocol)  
- 👛 Wallet Connection (Aptos Wallet Adapter)  
- ⚡ Fast & Modern UI (React + Vite / Next.js)  
- 🔒 Decentralized & Secure  

---

## 🛠️ Tech Stack

- **Frontend**: React / Next.js  
- **Blockchain**: Aptos  
- **Storage**: Shelby Protocol  
- **Wallet**: Petra / Martian / Fewcha  
- **Language**: TypeScript
